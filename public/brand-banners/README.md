# Brand banner images

Place horizontal banner images here to show above each brand’s product list.

## File names (by brand)

| Brand       | Filename        |
|------------|------------------|
| Higo       | `higo.jpg`      |
| Elf Bar    | `elf-bar.jpg`    |
| IVG        | `ivg.jpg`        |
| SKE        | `ske.jpg`        |
| Hayati     | `hayati.jpg`     |
| Pyne Pods  | `pyne-pods.jpg`  |
| Dojo       | `dojo.jpg`       |
| Accessories| `accessories.jpg`|

Use `.jpg` or `.webp`. Recommended size: **1200×300 px** (or similar 4:1 ratio).  
If a file is missing, the app shows a gradient header instead.
